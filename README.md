Calculator (Python Project)
Description

This is a simple text-based calculator built in Python. The program allows the user to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

The purpose of this project is to demonstrate fundamental programming skills, including user input, control flow, functions, and error handling.

Features
User input via terminal
Four basic operations (addition, subtraction, multiplication, division)
Error handling for invalid input
Protection against division by zero
Ability to run multiple calculations in a loop
Technologies Used
Functions (def)
Conditional statements (if, elif, else)
Loops (while)
Error handling (try/except)
Variables and return values
How to Run the Program
Open a terminal
Navigate to the folder where the file is located
Run the following command:
python kalkylator.py
Example Usage

The user selects an operation and inputs two numbers.
The program then returns the calculated result.

Purpose of the Project

This project demonstrates the ability to:

build a complete working program
structure code using functions
handle user input and errors
apply logical decision-making in code
